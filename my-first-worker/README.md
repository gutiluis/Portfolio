## Cloudflare worker

### The Fetch API provides an interface for asynchronously fetching resources via HTTP requests inside of a Worker.

### OAuth through login npx wrangler login

# does not run multiple or shared worker configs

### run the worker locally:
- npm create cloudflare@latest -- my-first-worker
- cd my-firs-worker
- npx wrangler login
- npx wrangler dev # run workers server for testing
- npx wrangler deploy # deploy worker code to the cloudflare network
